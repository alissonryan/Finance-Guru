# Cover Page

- Report Title
- Client / Stakeholder
- Prepared by (agent/persona)
- Date & data timestamp
- Compliance disclaimer (educational only, consult a licensed advisor)

# 1. Executive Summary

- Objective & scope
- Key findings (3 bullets)
- Recommended actions (3 bullets)
- Risk posture statement

# 2. Engagement Context

- User goals, constraints, time horizon
- Portfolio snapshot or problem statement
- Assumptions provided by client vs. researched

# 3. Market & Research Insights

- Macro backdrop (rates, volatility regime, liquidity)
- Security/sector findings with citations + timestamps
- Peer comparisons or benchmarks

# 4. Quantitative Analysis

- Metrics table (Sharpe, Sortino, drawdown, VaR 95/99)
- Scenario analysis (bull/base/bear) with narrative
- Sensitivity highlights (tornado or two-variable tables)

# 5. Strategy Recommendations

- Proposed tactics (margin/dividend/cash-flow/other)
- Alignment with user policies and constraints
- Benefits, risks, and monitoring triggers

# 6. Implementation Roadmap

- Step-by-step plan (timeline, owners, dependencies)
- Required artifacts (Excel model, presentation, compliance review)
- Tool usage plan ([run], [save:/path], etc.)

# 7. Risk & Compliance Review

- Margin buffer assessment (≥2× maintenance)
- Liquidity & diversification checks
- Regulatory/Tax considerations
- Limitations & open questions

# 8. Sources & Assumptions

- Table of data sources with timestamps and URLs
- Key assumptions (rates, fees, tax brackets)
- Notes on data quality or gaps

# 9. Appendices

- Detailed calculations or model excerpts
- Stress test outputs
- Glossary of terms/acronyms

# Footer

- Reiterate compliance disclaimers:
  - "Educational analysis; not personalized investment advice."
  - "Past performance does not guarantee future results."
  - "Consult a licensed advisor before acting."
- Contact / follow-up instructions
